# Práctica 1: Web scraping

## Descripción

Esta práctica se ha realizado bajo el contexto de la asignatura _Tipología y ciclo de vida de los datos_, perteneciente 
al Máster en Ciencia de Datos de la Universitat Oberta de Catalunya.

## Miembros del equipo

La actividad ha sido realizada de manera individual por **Juan Emilio Zurita Macías**.

## Ficheros del código fuente

* **src/main.py**: punto de entrada al programa. Inicia el proceso de scraping.
* **src/scraper.py**: contiene la implementación de la clase _ZooplaScraper_ cuyos métodos se encargan de extraer
datos de los diferentes listings publicados en la web [Zoopla](https://www.zoopla.co.uk).

## Recursos

1. Subirats, L., Calvo, M. (2019). Web Scraping. Editorial UOC.
2. Mitchel, R. (2015). _Web Scraping with Python: Collecting Data from the Modern Web_. O'Reilly Media, Inc. Chapter 1. 
Your First Web Scraper.
